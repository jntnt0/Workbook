Infrastructure Lab Portfolio - Jonathan Tant

This repository is a public infrastructure lab portfolio focused on systems administration, networking, cloud administration, and operational troubleshooting.

It documents hands-on work across Cisco CML, Azure, Microsoft 365, Intune, Windows Server, Linux, PowerShell, and general infrastructure operations. The goal is to show practical troubleshooting methodology, configuration notes, verification steps, rollback thinking, and runbook-style documentation.

⸻

Contents

Networking

The largest section of this repository. Covers routing, switching, WAN technologies, network security, and troubleshooting methodology.

Topics include:

* OSPF, EIGRP, BGP, and routing behavior
* VLANs, trunks, EtherChannel, STP/MST, and switching operations
* DMVPN, VXLAN, NAT, ACLs, VPNs, and firewall behavior
* Cisco CML lab documentation
* Verification, rollback, and failure analysis workflows

Notes are written in an operational format: mental model, configuration, verification, troubleshooting, rollback, and failure analysis.

Azure

Practical Azure infrastructure notes and labs focused on administration, deployment, and lifecycle management.

Topics include:

* Azure virtual machines
* Identity and access concepts
* Resource groups and infrastructure organization
* Terraform-based deployments
* Custom domain and TLS configuration
* IaaS/PaaS lifecycle management

Microsoft 365 and Endpoint Administration

Microsoft cloud administration notes tied to systems administration and endpoint governance.

* Operational runbook documentation
