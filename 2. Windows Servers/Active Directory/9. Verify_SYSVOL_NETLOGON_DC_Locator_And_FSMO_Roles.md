# Verify_SYSVOL_NETLOGON_DC_Locator_And_FSMO_Roles_Mental_Model
| Concept | Operational Meaning |
|---|---|
| SYSVOL | Domain controller share that stores Group Policy templates and logon scripts |
| NETLOGON | Domain controller share used for logon scripts and domain logon support |
| DC locator | Client/DC process for finding a domain controller through DNS SRV records |
| LDAP SRV | DNS service record used to find LDAP services on domain controllers |
| Kerberos SRV | DNS service record used to find Kerberos authentication services |
| Global Catalog SRV | DNS service record used to find Global Catalog servers |
| Netlogon service | Registers DC locator records and supports domain logon behavior |
| DFSR | Replication engine used for SYSVOL replication in modern domains |
| FSMO roles | Single-master AD operation roles required for specific domain/forest operations |
| Schema Master | Forest-wide FSMO role controlling schema updates |
| Domain Naming Master | Forest-wide FSMO role controlling domain naming changes |
| PDC Emulator | Domain-wide FSMO role used for time, password change priority, and legacy PDC behavior |
| RID Master | Domain-wide FSMO role allocating RID pools for security principals |
| Infrastructure Master | Domain-wide FSMO role maintaining cross-domain references |
| Global Catalog | DC role used for forest-wide searches and universal group membership lookup |
| First DC expectation | In a new forest, the first DC normally holds all FSMO roles and is a Global Catalog |
| First rule | Domain join may succeed once, but if SYSVOL, NETLOGON, SRV records, or FSMO state are broken, the domain is not healthy yet |

# Verify_SYSVOL_NETLOGON_DC_Locator_And_FSMO_Roles_Share_Verification_Skeleton
```powershell
# Run on the domain controller after promotion.

$DomainFqdn = "corp.local"
$DcName = "DC1"
$ValidationPath = "C:\ADPrep\post-promotion-validation"

New-Item -ItemType Directory -Force -Path $ValidationPath

# Confirm important services.
Get-Service NTDS,ADWS,KDC,Netlogon,DNS,DFSR |
  Tee-Object "$ValidationPath\core-services.txt"

# List local shares.
net share | Tee-Object "$ValidationPath\net-share.txt"

# Validate local SYSVOL and NETLOGON shares.
Test-Path "\\$DcName\SYSVOL"
Test-Path "\\$DcName\NETLOGON"

# Validate domain-based SYSVOL and NETLOGON paths.
Test-Path "\\$DomainFqdn\SYSVOL"
Test-Path "\\$DomainFqdn\NETLOGON"

# Validate local SYSVOL folder.
Test-Path "C:\Windows\SYSVOL"

# List SYSVOL contents.
Get-ChildItem "\\$DomainFqdn\SYSVOL" |
  Tee-Object "$ValidationPath\domain-sysvol-root.txt"

Get-ChildItem "\\$DomainFqdn\SYSVOL\$DomainFqdn" |
  Tee-Object "$ValidationPath\domain-sysvol-domain-folder.txt"

# Confirm Policies folder exists.
Get-ChildItem "\\$DomainFqdn\SYSVOL\$DomainFqdn\Policies" |
  Tee-Object "$ValidationPath\domain-sysvol-policies.txt"
```

# Verify_SYSVOL_NETLOGON_DC_Locator_And_FSMO_Roles_DNS_SRV_Skeleton
```powershell
# Run on the domain controller and optionally from a domain client.

$DomainFqdn = "corp.local"
$DcName = "DC1"
$DcFqdn = "$DcName.$DomainFqdn"
$DnsServer = "127.0.0.1"
$ValidationPath = "C:\ADPrep\post-promotion-validation"

New-Item -ItemType Directory -Force -Path $ValidationPath

# DC host record.
Resolve-DnsName $DcFqdn -Type A -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-dc-a-record.txt"

# LDAP DC locator records.
Resolve-DnsName "_ldap._tcp.dc._msdcs.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-ldap-dc-msdcs-srv.txt"

# LDAP domain records.
Resolve-DnsName "_ldap._tcp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-ldap-domain-srv.txt"

# Kerberos records.
Resolve-DnsName "_kerberos._tcp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-kerberos-tcp-srv.txt"

Resolve-DnsName "_kerberos._udp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-kerberos-udp-srv.txt"

# Password change records.
Resolve-DnsName "_kpasswd._tcp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-kpasswd-tcp-srv.txt"

Resolve-DnsName "_kpasswd._udp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-kpasswd-udp-srv.txt"

# Global Catalog records.
Resolve-DnsName "_gc._tcp.$DomainFqdn" -Type SRV -Server $DnsServer |
  Tee-Object "$ValidationPath\dns-gc-srv.txt"

# Optional site-specific lookup.
# Replace Default-First-Site-Name if custom AD sites are configured.
Resolve-DnsName "_ldap._tcp.Default-First-Site-Name._sites.$DomainFqdn" -Type SRV -Server $DnsServer -ErrorAction SilentlyContinue |
  Tee-Object "$ValidationPath\dns-site-ldap-srv.txt"
```

|---|---|---|---|

| GC SRV missing | DC is not Global Catalog or registration failed | `Get-ADDomainController -Filter *` | Confirm GC status and Netlogon registration |
