# Monitor_DHCP_Leases_Events_And_Scope_Utilization_Mental_Model
| Concept | Operational Meaning |
|---|---|
| Lease inventory | List of DHCP-assigned IP addresses and associated client identifiers |
| Scope utilization | Percentage of scope address pool currently used or available |
| Scope exhaustion | Condition where a scope has too few free addresses to reliably serve clients |
| Lease churn | High rate of lease creation/expiration that may indicate short lease duration, unstable clients, or network issues |
| Declined lease | Address rejected by a client, often because the IP is already in use |
| Bad address | DHCP conflict marker showing an address may already be used |
| Reservation audit | Review of fixed DHCP mappings for printers, servers, appliances, and known clients |
| Exclusion audit | Review of addresses DHCP must not lease |
| Scope option audit | Review of router, DNS, suffix, NTP, PXE, and other DHCP options |
| Failover health | State of DHCP failover partners and scope replication |
| DHCP audit log | Text log generated by DHCP Server for lease and service activity |
| DHCP event log | Windows event logs for DHCP Server admin and operational events |
| Threshold report | Report that flags scopes above warning or critical utilization |
| First rule | Monitor scopes before they are exhausted, not after clients start receiving APIPA |

# Monitor_DHCP_Leases_Events_And_Scope_Utilization_Lease_Inventory_Skeleton
```powershell
# Run in elevated PowerShell on the DHCP server.
# Purpose: export active lease inventory for all scopes and detect likely lease-state issues.

$EvidencePath = "C:\DHCP-Monitoring\Lease-Inventory"

New-Item -ItemType Directory -Force -Path $EvidencePath

Import-Module DhcpServer

$Scopes = Get-DhcpServerv4Scope
$AllLeases = @()

foreach ($Scope in $Scopes) {
  $ScopeId = $Scope.ScopeId.IPAddressToString
  $SafeScope = $ScopeId -replace '[\\/:*?"<>|]', '_'

  $Leases = Get-DhcpServerv4Lease -ScopeId $ScopeId -ErrorAction SilentlyContinue

  $Leases |
    Tee-Object "$EvidencePath\leases-$SafeScope.txt"

  $Leases |
    Export-Csv "$EvidencePath\leases-$SafeScope.csv" -NoTypeInformation

  $AllLeases += $Leases
}

# Export combined lease inventory.
$AllLeases |
  Export-Csv "$EvidencePath\all-leases.csv" -NoTypeInformation

$AllLeases |
  Tee-Object "$EvidencePath\all-leases.txt"

# Flag declined, bad, or unusual lease states if visible in returned properties.
$AllLeases |
  Where-Object {
    $_.AddressState -like "*Declined*" -or
    $_.AddressState -like "*Bad*" -or
    $_.AddressState -like "*Expired*" -or
    $_.AddressState -like "*Inactive*"
  } |
  Tee-Object "$EvidencePath\lease-state-risk-review.txt"

# Group leases by scope.
$AllLeases |
  Group-Object ScopeId |
  Sort-Object Count -Descending |
  Tee-Object "$EvidencePath\lease-count-by-scope.txt"

# Group leases by hostname when available to spot duplicate names.
$AllLeases |
  Where-Object {$_.HostName} |
  Group-Object HostName |
  Where-Object {$_.Count -gt 1} |
  Sort-Object Count -Descending |
  Tee-Object "$EvidencePath\duplicate-hostname-lease-review.txt"

# Group leases by client ID when available to spot multiple leases for same client.
$AllLeases |
  Where-Object {$_.ClientId} |
  Group-Object ClientId |
  Where-Object {$_.Count -gt 1} |
  Sort-Object Count -Descending |
  Tee-Object "$EvidencePath\duplicate-clientid-lease-review.txt"
```

# Monitor_DHCP_Leases_Events_And_Scope_Utilization_Scope_Utilization_Skeleton
```powershell
# Run in elevated PowerShell on the DHCP server.
# Purpose: monitor scope usage, free addresses, and high-utilization risk.

$EvidencePath = "C:\DHCP-Monitoring\Scope-Utilization"
$WarningPercent = 80
$CriticalPercent = 90

New-Item -ItemType Directory -Force -Path $EvidencePath

Import-Module DhcpServer

$Stats = Get-DhcpServerv4ScopeStatistics

# Export raw statistics.
$Stats |
  Tee-Object "$EvidencePath\scope-statistics.txt"

$Stats |
  Export-Csv "$EvidencePath\scope-statistics.csv" -NoTypeInformation

# Build threshold report.
$Report = $Stats | ForEach-Object {
  $PercentInUse = $_.PercentageInUse

  if ($PercentInUse -ge $CriticalPercent) {
    $Status = "CRITICAL"
  }
  elseif ($PercentInUse -ge $WarningPercent) {
    $Status = "WARNING"
  }
  else {
    $Status = "OK"
  }

  [PSCustomObject]@{
    ScopeId = $_.ScopeId
    Free = $_.Free
    InUse = $_.InUse
    PercentageInUse = $PercentInUse
    Reserved = $_.Reserved
    Pending = $_.Pending
    Status = $Status
  }
}

$Report |
  Sort-Object PercentageInUse -Descending |
  Tee-Object "$EvidencePath\scope-utilization-threshold-report.txt"

$Report |
  Sort-Object PercentageInUse -Descending |
  Export-Csv "$EvidencePath\scope-utilization-threshold-report.csv" -NoTypeInformation

# Export high-risk scopes only.
$Report |
  Where-Object {$_.Status -in @("WARNING","CRITICAL")} |
  Sort-Object PercentageInUse -Descending |
  Tee-Object "$EvidencePath\high-risk-scopes.txt"

# Suggested action output.
$Report |
  Where-Object {$_.Status -eq "CRITICAL"} |
  ForEach-Object {
    "CRITICAL: Scope $($_.ScopeId) is $($_.PercentageInUse)% used. Review lease duration, exclusions, reservations, stale leases, and scope expansion."
  } |
  Tee-Object "$EvidencePath\critical-scope-actions.txt"
```

```powershell
$EvidencePath = "C:\DHCP-Monitoring\Failover-Health"

$Failovers |
  Where-Object {
    $_.State -notlike "*Normal*" -and

    $_.Message -like "*partner*" -or
```

# Monitor_DHCP_Leases_Events_And_Scope_Utilization_Threshold_Report_Skeleton
```powershell
# Run in elevated PowerShell on the DHCP server.
# Purpose: generate a concise monitoring report that flags scope utilization risk.

$EvidencePath = "C:\DHCP-Monitoring\Threshold-Report"
$WarningPercent = 80
$CriticalPercent = 90
$ReportDate = Get-Date -Format "yyyy-MM-dd HH:mm:ss"

New-Item -ItemType Directory -Force -Path $EvidencePath

Import-Module DhcpServer

$ScopeStats = Get-DhcpServerv4ScopeStatistics
$Scopes = Get-DhcpServerv4Scope

$Report = foreach ($Stat in $ScopeStats) {
  $Scope = $Scopes | Where-Object {$_.ScopeId -eq $Stat.ScopeId}

  $Status = if ($Stat.PercentageInUse -ge $CriticalPercent) {
    "CRITICAL"
  }
  elseif ($Stat.PercentageInUse -ge $WarningPercent) {
    "WARNING"
  }
  else {
    "OK"
  }

  [PSCustomObject]@{
    ReportDate = $ReportDate
    ScopeId = $Stat.ScopeId
    ScopeName = $Scope.Name
    State = $Scope.State
    StartRange = $Scope.StartRange
    EndRange = $Scope.EndRange
    Free = $Stat.Free
    InUse = $Stat.InUse
    PercentageInUse = $Stat.PercentageInUse
    Reserved = $Stat.Reserved
    Pending = $Stat.Pending
    Status = $Status
  }
}

$Report |
  Sort-Object PercentageInUse -Descending |
  Format-Table -AutoSize |
  Tee-Object "$EvidencePath\dhcp-scope-threshold-report.txt"

$Report |
  Sort-Object PercentageInUse -Descending |
  Export-Csv "$EvidencePath\dhcp-scope-threshold-report.csv" -NoTypeInformation

# Separate warning and critical reports.
$Report |
  Where-Object {$_.Status -eq "WARNING"} |
  Export-Csv "$EvidencePath\warning-scopes.csv" -NoTypeInformation

$Report |
  Where-Object {$_.Status -eq "CRITICAL"} |
  Export-Csv "$EvidencePath\critical-scopes.csv" -NoTypeInformation

# Human-readable action list.
$Report |
  Where-Object {$_.Status -ne "OK"} |
  ForEach-Object {
    "$($_.Status): $($_.ScopeId) $($_.ScopeName) is $($_.PercentageInUse)% used. Free=$($_.Free), InUse=$($_.InUse)."
  } |
  Tee-Object "$EvidencePath\scope-utilization-action-list.txt"
```

# Monitor_DHCP_Leases_Events_And_Scope_Utilization_Failure_Checks
| Symptom | Likely Layer | First Check | Do Not Start With |
|---|---|---|---|
| Scope near exhaustion | Too many active leases, long lease duration, too many exclusions, or scope too small | `Get-DhcpServerv4ScopeStatistics` | Rebooting DHCP server |
| Clients getting APIPA | Scope exhaustion, relay issue, DHCP down, authorization issue | Scope stats, service, authorization, relay path | Editing DNS options |
| Many declined leases | IP conflicts or duplicate static addresses | Audit logs and lease state | Expanding scope first |
| BAD_ADDRESS entries appear | Conflict detection or duplicate IP use | Audit logs and active leases | Deleting random reservations |
| Lease count spikes | Short lease duration or unstable client population | Lease duration, audit logs, client VLAN | Rebuilding server |
| Scope stats unavailable | DHCP module/service issue or permissions | `Get-Service DHCPServer`; admin context | Reinstalling DHCP |
| Audit logs missing | Audit logging disabled or wrong path | `Get-DhcpServerAuditLog` | Assuming no DHCP activity |
| Event logs empty | Wrong log, service inactive, or event retention | DHCP Admin/Operational/System logs | Clearing logs |
| Scheduled report does not run | Scheduled task permissions or execution policy | Task history and script path | Editing scopes |
| Report CSV empty | DHCP module not loading under task account | Scheduled task action and account context | Deleting task immediately |
| Failover state unhealthy | Partner communication or replication issue | `Get-DhcpServerv4Failover`; partner reachability | Removing relationship first |
| Partner scope differs | Replication drift | `Invoke-DhcpServerv4FailoverReplication`; compare options | Manually editing partner |
| Reservations look duplicated | Same MAC or IP reserved in multiple scopes | Combined reservation audit | Deleting all reservations |
| Exclusions consume too much pool | Exclusion ranges too broad | Exclusion audit and scope statistics | Expanding subnet blindly |
| DNS records stale | DHCP DNS cleanup or scavenging issue | DHCP DNS settings and DNS record timestamps | Clearing DHCP leases only |
| Client has lease but wrong gateway | Scope option 003 wrong | `Get-DhcpServerv4OptionValue -ScopeId <scope-id>` | Rebuilding relay |
| Client has lease but wrong DNS | Scope option 006 wrong | Scope options | Changing failover |
| Rogue DHCP suspected | Client gets unexpected DHCP server | Client `ipconfig /all` DHCP Server field | Editing authorized server only |
| DHCP service stops | Service crash, database issue, event log evidence | System and DHCP Admin logs | Recreating scopes |
| High utilization but few clients visible | Reservations/exclusions reduce free pool | Scope statistics plus reservations/exclusions | Deleting leases randomly |
