ASAv_ICMP_Trace_Interface_Fault_Diagnosis_Mental_Model

| Concept | Operational Meaning |
|---|---|
| ICMP Echo (ping) | Tests basic end-to-end reachability; sends ICMP Echo Request and expects ICMP Echo Reply |
| pathping | Windows tool combining ping and traceroute; shows each hop and where responses stop |
| Request timed out | No response received; packet is being dropped silently or the interface is completely non-functional |
| Destination host unreachable | An intermediate device actively returned an ICMP unreachable; path was reached but destination was not |
| Last described hop | In pathping output, the last hop returning a name or IP rather than only `* * *`; this is where the fault is localized |
| ASA interface nameif | The logical name assigned to a physical interface; required before the interface participates in routing or policy |
| DHCP identifier formula | Cisco ASA uses `<MAC><interface_name>-<host>` to request an address from DHCP; missing nameif breaks the formula and no IP is assigned |
| Interface without nameif | An unconfigured ASA interface will not pass traffic even if the physical link is up |
| ASDM Interfaces panel | Configuration > Device Setup > Interface Setup > Interfaces; shows all interfaces and their nameif, IP, and enabled state |

ASAv_ICMP_Trace_Interface_Fault_Diagnosis_Configuration_Checklist

| Step | Task | Device | Action | Expected Result |
|---:|---|---|---|---|
| 1 | Attempt initial ping to IIS_Server | Win_Station | Run `ping IIS_Server` in CMD/PowerShell | Request times out; no reply received |
| 2 | Run pathping to IIS_Server | Win_Station | Run `pathping IIS_Server` in CMD/PowerShell | Win_Station is last described hop; all hops beyond show only `* * *` |
| 3 | Note fault localization | Win_Station | Identify last hop with a name or IP in pathping output | Fault is on the hop immediately after Win_Station: the ASAv Workstations interface |
| 4 | Open ASDM | Win_Station | Launch ASDM from desktop shortcut; connect to `mgmt.asav.internal` as `admin` / `P@ssw0rd` | ASDM loads and displays the device dashboard |
| 5 | Navigate to interface list | ASDM | Configuration > Device Setup > Interface Setup > Interfaces | All ASAv interfaces listed with current configuration |
| 6 | Identify the unconfigured interface | ASDM | Compare TenGigabitEthernet interfaces; locate the one with no name and no IP | TenGigabitEthernet0/0 has no nameif and no IP; all others are configured |
| 7 | Edit the interface | ASDM | Double-click TenGigabitEthernet0/0 | Interface edit dialog opens |
| 8 | Assign the interface name | ASDM | In the Interface Name field (row 3, first tab), type `Workstations` | DHCP identifier formula completes as `<MAC>Workstations-<host>` |
| 9 | Save and apply | ASDM | Click OK, then click Apply | Configuration pushed to ASAv; interface requests a DHCP address |
| 10 | Confirm IP assignment | ASDM | Return to Interfaces panel; verify TenGigabitEthernet0/0 now shows an IP | Interface has obtained an IP from DHCP |
| 11 | Retest connectivity | Win_Station | Run `ping IIS_Server` in CMD/PowerShell | Ping succeeds; ICMP Echo Reply received from IIS_Server |

ASAv_ICMP_Trace_Interface_Fault_Diagnosis_Failure_Checks

| Symptom | Likely Cause | Device | Command | Fix |
|---|---|---|---|---|
| pathping shows all `* * *` beyond Win_Station | ASAv Workstations interface has no nameif or IP | Win_Station | `pathping IIS_Server` | Assign nameif `Workstations` to TenGigabitEthernet0/0 in ASDM |
| Interface shows in ASDM but has no IP after naming | DHCP server unreachable or interface not enabled | ASDM | Configuration > Interfaces panel | Verify interface is enabled and DHCP server is reachable on the segment |
| Ping still fails after applying nameif | Changes not pushed to device | ASDM | Click Apply and confirm no error banner | Ensure Apply completed without error |
| ASDM Apply returns error on nameif change | Name conflicts with an existing interface | ASDM | Review interface list for duplicate names | Use a unique nameif value |
| Ping succeeds then immediately stops | DHCP lease expired and not renewed | Win_Station / ASDM | Check IP on Workstations interface in ASDM | Investigate DHCP server availability on the workstation segment |
