# Splunk_Suspicious_Pair_Byte_Volume_Analysis_Mental_Model

| Concept | Operational Meaning |
|---|---|
| `sum(bytes_in)` | Totals all inbound bytes received by `src_ip` from `dest_ip` across all log events for that pair |
| `sum(bytes_out)` | Totals all outbound bytes sent from `src_ip` to `dest_ip` across all log events for that pair |
| `sort - total_bytes_out` | Sorts descending by outbound volume — highest outbound transfer surfaces first |
| Separating bytes_in and bytes_out | Directional split reveals whether traffic is symmetric (normal) or heavily skewed outbound (exfiltration indicator) |
| Combined total | `total_bytes_in + total_bytes_out` gives the full bidirectional data volume for the pair |
| Exfiltration byte signature | A heavily skewed `bytes_out` value relative to `bytes_in` is consistent with data being pushed out to an external destination |

# Splunk_Suspicious_Pair_Byte_Volume_Analysis_Investigation_Checklist

| Step | Action | Notes |
|---|---|---|
| 1 | Confirm time range is set to **All Time** | Consistent with Task 1 — partial time range will undercount totals |
| 2 | Run base query to get byte totals for all IP pairs | Confirms `total_bytes_in` and `total_bytes_out` columns populate |
| 3 | Identify the top result by `total_bytes_out` | The suspicious pair from Task 1 should surface at or near the top |
| 4 | Record `total_bytes_in` value for the suspicious pair | Represents inbound volume (~2,523,690) |
| 5 | Record `total_bytes_out` value for the suspicious pair | Represents outbound volume (~82,530,430) |
| 6 | Add `total_bytes_in + total_bytes_out` | Combined bidirectional total = **85,054,120** |
| 7 | Note the directional skew | Heavily outbound-weighted confirms active data exfiltration pattern |

# Splunk_Suspicious_Pair_Byte_Volume_Analysis_Base_Query_Skeleton

```spl
source="<DATA_SOURCE>"
| stats sum(bytes_in) as total_bytes_in sum(bytes_out) as total_bytes_out by src_ip dest_ip
| table src_ip dest_ip total_bytes_in total_bytes_out
| sort - total_bytes_out
```

# Splunk_Suspicious_Pair_Byte_Volume_Analysis_Failure_Checks

| Symptom | Likely Cause | Fix |
|---|---|---|
| No results returned | Time range not set to All Time | Reset time picker to All Time |
| Query syntax error after paste | Encoding issue — minus sign or quotes malformed | Manually retype `sort - total_bytes_out` and quote characters |
| Suspicious pair not at top of results | Sort applied to `bytes_in` instead of `bytes_out` | Confirm `sort - total_bytes_out` is correctly specified |
| Combined total does not match answer choices | Only one directional field was summed | Ensure both `total_bytes_in` and `total_bytes_out` are added together |
